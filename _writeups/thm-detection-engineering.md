# TryHackMe: Detection Engineering Challenge

## Overview

This exercise combines offensive behavior with defensive analysis. It focuses on translating reconnaissance and exploitation patterns into SIEM rules and monitoring coverage.

## Highlights

- Tracked anomalous PowerShell execution
- Built detection logic for suspicious process chains
- Validated alerts against simulated adversary activity

## Learnings

Understanding attacker tradecraft is essential to improve detection fidelity and reduce alert noise in a live SOC environment.
